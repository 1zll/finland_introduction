# 概要


## できること
- フィンランドのおすすめスポット5選がどんな場所か知れる。

## 特徴
- フィンランドの国旗の色である濃い青色をメインに使った。
- 全編英語で構成。

## 作成理由
Figmaを使ってデザインを考えてみたかった。
折角作ったデザインをそのままにしておくのも勿体ないなと思い1年生の頃だったので、HTML/cssの練習として制作した。
※初めてデザインからコーディングまで一貫して作成した作品です。
